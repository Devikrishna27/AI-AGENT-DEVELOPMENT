# Productivity AI Agent

## Overview
This repository contains a simple Python-based Productivity AI Agent. The agent interacts with users through text input and provides productivity-related assistance such as study plans, to-do lists, and motivational messages. The project demonstrates the basic working of an AI agent using rule-based logic.

## Features
- Interactive command-line AI agent
- Generates simple study plans
- Helps create to-do lists
- Provides motivational quotes
- Continuous interaction until user exits

## How the AI Agent Works
The AI agent uses:
- A predefined list of motivational quotes
- Conditional statements (`if-else`)
- A loop to continuously interact with the user

Based on keywords in the userâ€™s input (such as *study plan*, *to-do*, or *motivate*), the agent generates an appropriate response.
